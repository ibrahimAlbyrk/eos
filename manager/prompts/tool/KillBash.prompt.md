---
description: "Built-in tool — KillBash (alias of KillShell)"
---
Kills a running background shell that was started with the Bash tool's `run_in_background` parameter.

Usage:
- Provide the `shell_id` of the background shell to terminate.
- Returns a confirmation once the shell has been stopped, or an error if no background shell with that id exists.
- KillBash is an accepted alias of KillShell; both name the same shell-termination operation.
